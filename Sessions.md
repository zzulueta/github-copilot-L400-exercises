L400 Advanced GitHub Copilot Sessions

Overview:
This series of three advanced sessions is designed for experienced developers and team leads looking to deepen their expertise in GitHub Copilot. Each session focuses on practical applications of Copilot in real-world scenarios, enhancing code quality, review processes, and team collaboration.

Session 1:
Understand inline, chat, and contextual scopes in Copilot.
Optimise Copilot performance by managing file- and repo-level context.
Create and apply Copilot Instructions for organisational style and standards.
Use Copilot Chat commands (/fix, /explain, /tests, /optimise) for debugging and refactoring.
Apply Copilot for inline documentation, test coverage, and clean commit hygiene.

Session 2:
Conduct contextual code reviews using Copilot Chat.
Build a custom review chat mode tuned to team workflows and standards.
Integrate repository context (README, style guide, PR templates) into review sessions.
Use Copilot Tools for intelligent search, documentation lookup, and inline review support.
Apply conversational feedback loops to improve review depth and developer engagement.

Session 3:
Use Copilot Chat to establish lightweight governance and peer review processes.
Reinforce consistency and quality through conversational review and summary tools.
Apply Copilot for auto-documentation, PR summaries, and knowledge sharing.
Execute a full end-to-end quality workflow: refactor → review → document.
Present outcomes demonstrating Copilot’s role in CBA’s internal governance model.
